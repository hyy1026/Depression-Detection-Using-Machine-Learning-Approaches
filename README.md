# Depression-Detection-Using-Machine-Learning-Approaches
A Hybrid Energy-Attention Transformer (HEAT) for text-based depression detection using Transformer, TF-IDF priors, and energy-based inference.

## Dataset
The datasets used in this project have been preprocessed and split into training, validation, and testing sets to ensure reproducibility.
Download the complete dataset from Kaggle: https://kaggle.com/datasets/ee25765f37847f78000ef7aecdb36a795539cf8cd8672c0f6e314edd48254d41

The provided dataset includes:
- `train.csv` – Training dataset used for model training.
- `validation.csv` – Validation dataset used for hyperparameter tuning and early stopping.
- `test.csv` – Test dataset used for the final performance evaluation.
